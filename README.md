# Simon-says-game
Simon says game has four colored buttons, each blinking when it is pressed or activated by the device.
🎮 Simon Says Game
A simple and interactive browser-based game inspired by the classic Simon Says memory challenge. This project was built as part of my learning journey in HTML, CSS, and JavaScript.

Purpose:-
The goal of the game is to test the user’s memory. At each level, the game highlights a sequence of colored buttons in a particular order. The player must repeat the exact sequence by clicking the buttons in the correct order. The sequence grows by one step every level.

This project helped me practice:
DOM manipulation
Event handling
Arrays and sequences
Game logic

Features:-
Press any key to start the game
Randomly generated color sequence that increases in length each level
Visual button flash animations
Real-time input validation
Score tracking and game-over message
Reset and replay functionality

Technologies Used :-
HTML: To structure the game layout (title, instructions, buttons).
CSS: For layout styling, button coloring, and visual effects.
JavaScript: To manage game logic, track sequences, handle user input, and provide feedback.

What I’ve Learned:-
As a beginner, I’ve learned the following by building this project:

How to listen for keyboard and click events
Use of arrays to store and compare sequences
How to manipulate the DOM dynamically
Implementing delays using setTimeout
The importance of clear variable naming and code readability
Basics of modularizing logic using functions (levelUp(), flash(), correct(), etc.)
